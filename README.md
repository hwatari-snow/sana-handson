# ❄️ Snowflake ハンズオン

🚚 Tasty Bytes（架空のフードトラック会社）のデータを使って、Snowflakeの機能を体験してみよう！

## 📚 このハンズオンで学べること

### 1. ⚙️データパイプライン
JSONデータを取り込んで、メニューの原材料を分析するパイプラインを作ります。
- 半構造化データ（JSON）からデータを抽出
- Dynamic Tableで自動的にデータを更新

### 2. 🤝 データコラボレーション
外部データと自社データを組み合わせて、新しい分析をしてみます。
- MarketPlaceから気象データを取得
- お隣同士でデータシェアリング

### 3. 📈 BI＆AI
顧客ロイヤルティを可視化するダッシュボードを作ります。
- Streamlitで顧客分析ダッシュボード
- Snowflake IntelligenceでAIへチャット分析

## 🚀 やってみよう

1. `setup.sql`を実行して環境をセットアップ
2. ハンズオン1でデータパイプラインを作る
3. ハンズオン2で外部データを取り込む
4. ハンズオン3でダッシュボードを作る、Cortex Agentの作成
